# Applied Machine Learning Portfolio

This repository presents a selection of **machine learning projects developed in Python** as part of my academic training in the **Data Science Master's program at UOC**.

The portfolio is designed to highlight practical work across the main stages of applied machine learning, including **unsupervised learning**, **supervised learning**, and **ensemble modelling**. The projects combine data preparation, exploratory analysis, model selection, evaluation, and interpretation of results.

## Repository purpose

This repository is intended as a compact portfolio for recruiters, hiring managers, and technical professionals interested in my background in:

- Machine learning
- Data science
- Predictive analytics
- Model evaluation
- Python-based analytical workflows
- Applied problem solving with real datasets

## Projects included

I recommend using the following folder names to make the repository clearer and more professional:

### 1. `01_unsupervised_learning_recipes`
Project focused on **unsupervised learning methods** applied to recipe data. It reflects work related to pattern discovery, similarity analysis, and structure extraction from unlabeled data.

### 2. `02_supervised_learning_applications`
Project focused on **supervised learning methods**, using applied classification and predictive modelling exercises with datasets related to ransomware detection and videogame data.

### 3. `03_ensemble_models_wind_turbine`
Project focused on **ensemble learning and model combination**, applied to wind turbine data for predictive analysis and model performance improvement.
